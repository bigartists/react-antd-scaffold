# react 脚手架

#### 介绍

基于create-react-app 的一个脚手架，总体没什么太多亮点；
支持vite，webpack； 




#### 安装与使用

> 如果没有pnpm命令，可使用npm 或者 yarn

1.  pnpm install
2.  pnpm run start 
3.  pnpm run build 

#### 参与贡献

1.  Fork 本仓库到个人仓库
2.  git checkout -b dev upstream/dev
3.  提交代码到个人仓库 git push origin dev
4.  新建 Pull Request
